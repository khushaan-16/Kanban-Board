# 📌 Kanban Board App

A simple and interactive **Kanban Board** built using **HTML, CSS, and JavaScript**.
It allows users to create, drag, drop, and manage tasks across different stages.

---

# 👉 Play the game here:
🔗 [https://kanban-boardkb.netlify.app/]

---

## 🚀 Features

* ✅ Create new tasks with title & description
* 🗂️ Organize tasks into:

  * To Do
  * In Progress
  * Done
* 🖱️ Drag & drop tasks between columns
* ❌ Delete tasks instantly
* 💾 Persistent data using **localStorage**
* 🔢 Task count per column updates dynamically
* 🎯 Clean and minimal UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Browser Local Storage

## ⚙️ How It Works

* Tasks are stored in an object and synced with **localStorage**
* Drag-and-drop functionality is implemented using:

  * `dragstart`
  * `dragover`
  * `drop`
* Each column dynamically updates:

  * Task list
  * Task count
* Data persists even after page reload
